A design pattern describes a solution to a design problem.

The intent of a design pattern is to provide a description which allows a designer to determine the following:
  1. can the pattern be used in the current project
  2. can the pattern be re-used
  3. can the pattern be used as a guide to develop similar patterns that may be structurally or functionally different
